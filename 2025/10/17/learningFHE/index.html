1. Number Theory and Cryptographic Hardness Assumptions数论和密码学困难假设

现代密码学系统总是基于一个困难问题提出的，例如：RSA基于大整数因子分解问题、Diffie-Hellman基于有限域上的‌离散对数问题‌、ECC基于椭圆曲线上的离散对数问题等等…

引出One-way Function单向函数，以及基于何种大数困难问题。 → 是否就是引出One-way Trapdoor Function单项陷门函数？

2. 基础数论知识预备

后续是一些简单的数论问题在密码学中的运用

2.1. 素数和可除性

  1. a|b → a 整除 b → b=ka
    同时若 a∉{1,b} 那么 a 称为 b 的一个非平凡除数 → 1和它本身称为平凡约数；大于1小于它本身的约数称为非平凡约数。整数b的非平凡约数(nontrivial divisor)称为b的因数(factor)。
    a 是一个整数，设 b 是一个正整数，那么存在唯一的整数 q、r ，使得 a=qb+r 并且 0≤r<b 。

  2. 最大公约数 c 记作 gcd(a,b) ，使得 c|a 和 c|b 。如果 gcd(a,b)=1 ，a和b是互质的。
    设 a,b 为正整数，则存在 X,Y 使得 Xa+Yb=gcd(a,b) 。此外， gcd(a,b) 是能以这种方式表示的最小正整数。

  3. c|ab 和 gcd(a,c)=1 ，那么 c|b 。因此，如果 p 是素数且 p|ab ，则要么 p|a or p|b 。

  4. 如果 a|N ， b|N ，和 gcd(a,b)=1 ，那么 ab|N 。

2.2. 模运算

同余关系： a 和 b 模 N 同余，记作 $ a=b \bmod N$ 即，即 [amodN]=[bmodN]
同余有自反性，传递性,同余也遵守关于加法、减法和乘法的标准算术规则。注：无除法， ab=cbmodN 不一定意味着 a=cmodN 。

3. 素数、分解和RSA

正如前面所说，这里引出RSA，RSA是基于大整数分解困难问题提出的公钥。原文引出整数分离困难，再紧接着提出最难分解的数是那些只有大素数因子的数。

3.1. 生成随机素数

找n位中的素数，用循环一次一次找n位随机数，对比是否为素数。

引申出素数定理，数学中的一个重要结果，给出了给定长度的整数中素数比例的相当精确的界限。
对于任何 n>1 ， n 位整数为素数的比例至少为 1/3n 。

设置t=3n2，那么在所有t次算法迭代中，一个素数没有被选中的概率最多是

(1−13n)t=((1−13n)3n)n≤(e−1)n=e−n

在n中是可忽略的。因此，使用poly(n)次迭代，得到一个算法，其输出失败的概率在n中是可忽略的。

3.2. 素性测试

如果p是素数，那么Miller–Rabin测试总是输出素数。如果p是合数，算法除以概率至多为2^(-t)外，总是输出合数。
