c语言初体验

一.关于c语言的框架
#include <文件名>
int main()
{
return 0;
}
这样就是一个基础的c语言框架，目前接触不多，了解甚少，希望日后能更多的接触不同的框架。
二.关于关于stdio.h中六种常用方法的小结
scanf函数–格式化输入函数
printf函数–格式化输出函数
getchar函数–从缓冲区读取一个字符，但只有第一个字符作为返回值，该函数的输入知道回车才结束，但从键盘输入一堆字符时，这些字符会存储在缓冲区中，按下回车键开始调用getchar函数，把最先存储的字符读取，然后删除，在继续读取释放，直到满足条件为止。
putchar函数–功能：输出一个字符
gets函数–功能：从缓冲区读取一个“字符串”存储到字符指针变量str指向的内存空间
puts函数–功能：将字符串输出到终端，puts函数一次只能输出一个字符串，字符串可以包含转义字符
注意：
putchar函数只能用于单个字符；
当putchar参数为被单引号括起来的字符时，直接输出该字符；
当参数介于0~127时且为整数时，视为ASCII码，输出该ASCII码对应的字符；
当参数为事先定义好的char变量时，输出该变量指向的字符。
