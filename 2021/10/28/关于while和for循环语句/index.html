关于while和for循环语句

发表于 2021-10-28 更新于 2021-11-09

关于while
while，中文叫 当…时，其实就是当条件满足时就执行代码，一旦不满足了就不执行了。
while (条件) { 满足条件就执行 }
关于for
for作为一种较为常用的循环语句，经常出现于题目中。相比于while，可能更难于理解。
具体操作可以像这样：
for(i=1,i<10,i++){
要执行的语句
}
这个控制语句中，i=1是循环的开始，我们给他定义的一个量。
i<10是循环的最后要满足的条件，一旦不符合就结束循环。
i++是循环一次后进行步骤，就好比i=1带入要执行的语句中得到了个结果，在第二次循环中它变成了i++,在第二次循环开始前i=2。
可能这个循环语句大家一开始接触时会觉得十分的抽象，但是只要把它分为三个部分：开始时，循环最后满足的条件，一次循环后所要执行的。这三个部分会更有利于我们去理解这个程序。

举个例子，像作业里面的求级数之和：e^x的泰勒展开式。

#include <stdio.h>
#include <math.h>
int main()
{
double sum=1,p,i,e=1,x;
int n;
printf(“请分别输入n,x：\n”);
scanf(“%d %lf”,&n,&x);
if(n<0){
printf(“错误”);
return 0;
}
else{
for(p=1;p<=n;p++){
i=pow(x,p);
e=e*p;
sum+=(i/e);
}
printf(“e^%.1lf=%.1lf”,x,sum);
}
return 0;
}
在这个循环中就可以明显的看出p=1是开始的值，p<=n是结束时需要满足的条件，p++是这个循环过程中所需要执行的语句。
目前阶段而言，for循环较为常用而且只要理解了它这三个部分，for循环就会变得很简单。
当然这还只是我个人的见解。
